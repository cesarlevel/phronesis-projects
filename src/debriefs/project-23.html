Project 23: Integrate Checkout.com gateway

What made you smile?

Very easy to integrate the gateway.

What did you find confusing?

Finding the right information no the Checkout.com documentation.

  • Initially, the gateway integration failed due to the Secret API key not being configured with the processing channel ID. However, upon further review of the documentation, we successfully integrated the gateway by following the same steps used for services like Zoura. Notably, no additional settings were required beyond providing the secret API key during the Recomm gateway configuration.
