What did you find confusing?

The segment creation process was actually quite simple.

  • CONFUSION When trying to edit a segment, there was always some confusion between the link to the segment's name and the edit button (pencil icon).
  • BUG When copying a segment URL with a number range filter ending in two dots (e.g., 100..), the URL becomes invalid. This happens because messaging apps' URL validation regex patterns filter out these dots, causing shared data tables to appear empty.
